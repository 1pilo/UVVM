- bitvis_vip_wishbone                     : An example VIP that uses UVVM
  - doc                                   : Quick Reference for BFM and VVC
  - script                                : Compile scripts
  - src                                   : Wishbone BFM and VVC source code

IMPORTANT
This is a simplified Verification IP (VIP) for Wishbone.
The given VIP complies with Wishbone for single accesses, and thus allows a normal single access towards an Wishbone interface.
This VIP is not an Wishbone protocol checker.
For a more advanced VIP please contact Bitvis AS at support@bitvis.no
